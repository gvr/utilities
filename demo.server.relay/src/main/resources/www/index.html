Example

To test the application use for example the following link to subscribe (opens in a new screen): api/testUser/testAction. Then execute curl commands with messages.
curl -X POST -H "Content-Type: application/json" -d '{"id":"1","user":"testUser","action":"testAction","body":{"bla":"testjsonobject","zzz":[1,3,3]}}' http://localhost:8080/relay
Every user/action combination is pushed to a connection opened as api/<user>/<test>.
